# Anchor Manifest

This file lists the key components and files ("anchors") of the project.

## Core Components
- `src/App.jsx`: Main application entry point.
- `src/components/QuestionItem.jsx`: Displays individual questions.
- `src/components/ReviewMode.jsx`: Interface for reviewing questions.
- `src/components/AnalyticsDashboard.jsx`: Dashboard for viewing metrics.

## Services
- `src/services/gemini.js`: Gemini API integration.
- `src/services/googleSheets.js`: Google Sheets integration.

## Hooks
- `src/hooks/useQuestionManager.js`: Manages question state.
- `src/hooks/useGeneration.js`: Handles question generation logic.
