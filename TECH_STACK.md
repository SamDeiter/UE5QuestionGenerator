# Tech Stack & Constraints
- **Framework**: React 18 (Functional components only, no Class components)
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS (Use utility classes, avoid custom CSS files)
- **Backend**: Firebase v9 (Modular SDK, NOT namespaced)
- **Testing**: Vitest + React Testing Library (Do not use Enzyme)
- **State**: Custom Hooks (e.g., `useQuestionManager`), no Redux/Zustand.
